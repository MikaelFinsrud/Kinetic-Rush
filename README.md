# Kinetic Rush

Kinetic Rush is a 3D first-person precision platformer prototype built in Unity, focused on responsive movement, momentum, and clean system design.

Unlike my earlier projects, this prototype was developed with a strong emphasis on code quality, modularity, and maintainability.

---

## 🎮 Core Concept

The game revolves around momentum-based movement using push and pull mechanics on metal objects and thrown anchors, combined with precise ground movement and controlled air adjustments.

---

## 🚀 Features

* Physics-based push and pull movement mechanics
* Momentum-preserving sliding and slide-jumping
* Responsive and snappy ground controls
* Limited but meaningful air control
* Modular and extensible movement system

---

## 🛠️ Tech Stack

* **Engine:** Unity (C#)
* **Architecture:** Component-based, modular systems
* **Version Control:** Git + GitHub

---

## 🧠 Focus & What I Learned

* Writing clean, maintainable, and modular C# code
* Designing systems with clear responsibilities
* Improving code structure compared to earlier projects
* Iterating on gameplay while maintaining code quality
* Balancing flexibility and simplicity in system design

---

## 🧱 Code & Architecture

This project was built with a deliberate focus on:

* Separation of concerns
* Readable and self-documenting code
* Reusable components and systems
* Avoiding tight coupling between systems

Compared to previous projects, this represents a significant improvement in how I structure and maintain code.

---

## 🎯 Purpose of the Project

The goal of Kinetic Rush was to improve my programming skills and code quality, while experimenting with movement systems in a small, controlled prototype.
